	COOLS TRICKS AND THINGS YOU CAN DO WITH THE WINDOWS CMD
-> mkdir <dir_name>
-> Cipher /E
-> help color <display the list of colors in CMD>
-> attrib +h +s +r (hides a folder not to be visible)
-> attrib -h -s -r (unhides a folder be visible)
-> title <name_of_the_new_title> (change the name of the title of the Command Prompt)
-> ipconfig | clip (copies the files from a specified location inside your terminal to the clipboard so you can paste it somewhere else)
-> wmic product get name (list all the programs that are installed in your PC)
-> press F7 to get a list of commands you have used in opening different applications inside a different section
